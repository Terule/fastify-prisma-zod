# fastify-prisma-zod
A simple backend template with minimal configuration. It includes Fastify, Swagger documentation, Zod and Prisma ORM.

### dependencies:
- fastify
- zod
- @fastify/cors
- @fastify/swagger
- @fastify/swagger-ui
- @prisma/client
- fastify-type-provider-zod

### devDependencies:
- @eslint/js
- @types/node
- eslint
- globals
- typescript
- prisma
- tsx
- typescript-eslint
